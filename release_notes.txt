## v1.0.1 — Crash Fix

Fixes crash on ARM64 Android 14+ when hooking target apps.

**What was wrong:**
1. `__builtin___clear_cache` on GOT data addresses triggered MTE/tagged-pointer abort in libc
2. `mprotect` return value was unchecked — if page couldn't be made writable (SELinux/seccomp), write caused SIGSEGV

**What changed:**
- Removed `__builtin___clear_cache` call (not needed for GOT data writes; mprotect barriers suffice)
- Check `mprotect` return value; skip GOT entry gracefully on failure
- Null-guard on `orig_prop_get` before forwarding calls
- Mount hiding re-enabled
